# vangtichai

A responsive Flutter currency breakdown calculator built for a class assignment. The application instantly calculates the exact combination of Bangladeshi Taka notes required for any input amount.

## Features
* **Custom Block Keypad:** Built-in plain $0-9$ keypad with a dedicated `CLEAR` button.
* **Instant Calculation:** Uses efficient integer division and modulo operations to break down currency amounts dynamically.
* **Supported Denominations:** 500, 100, 50, 20, 10, 5, 2, 1.
* **Fully Responsive:** Adapts its interface cleanly between Portrait (vertical) and Landscape (horizontal) orientations without screen overflow issues. Works seamlessly across both phones (e.g., Pixel) and tablets (e.g., Pixel Tablet / Nexus 10).

## How to Run
1. Ensure Flutter and Android SDK are fully installed on your machine.
2. Clone or download this repository.
3. Open the project directory in VS Code or Android Studio.
4. Launch an emulator or connect a device.
5. Run `flutter run` in your terminal.
